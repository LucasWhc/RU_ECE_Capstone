Music Transcription

Project for everyone to create their own music

In this project, we created an application that can make music sheet automatically with a given part of audio instead of writing it manually, which saves time and doesn't require expertise.

overview

We divided the whole project into 3 parts, shown as following

Audio Record

In this part, we play a part of an audio and let the computer record it. Then we save it into a wav file, whose features will be extracted.

Music Transcript

In this part, we use the audio we collected from the previous part to do the transcription by utilizing its spectrogram and chromagram.

Output MIDI

After the transcription, we have several numerical matrix representing features of frequency, which we can use to create a MIDI file.

how it works

In this part, we will describe the detailed principle of our project.

Q1: Why do we want to do it?

There are already existing software that allow people to write music sheet manually. However, there is few apps that can finish music transcription automatically. Existing works is time-consuming and require expertise on music, which most people lack of. Our apps can save people's time and provide help for those who do not know much about but is highly interested in music.

Q2: How do we extract features?

First we record the given audio into a wav file. As we know, in music, different pitches represent different frequencies. We first draw the chromagram and spectrogram of this wav file. Then, with a sliding window, we combine the chroma feature and the spectrogram of the music together to extract digital values showing the relationship between time and frequency.

Q3: How do we utilize the extracted features?

We took advantage of the library mido in Python. In this library, each note has its own number, which can be calculated by frequency. In the previous step, we already know the features of time and frequency. Based on that, we can calculate the number, which will be transformed to a note and the time features represent how long each note lasts. Then we have all the elements we need to create a MIDI.

Q4: What is MIDI?

MIDI is a technical standard that describes a communications protocol, digital interface, and electrical connectors that connect a wide variety of electronic musical instruments, computers, and related audio devices for playing, editing and recording music. To make it easier to understand, MIDI is the music sheet which can be read and understood by the computer.
